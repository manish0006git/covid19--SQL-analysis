# 🦠 COVID-19 Data Analysis with SQL

## Overview
End-to-end SQL analysis of global COVID-19 data covering cases,
deaths, and vaccinations across 200+ countries (2020–2023).

## Skills Demonstrated
- Complex JOINs across multiple tables
- Window Functions (SUM OVER, AVG OVER, RANK)
- CTEs (Common Table Expressions)
- Aggregations, CASE WHEN, NULLIF
- Views for reusable logic
- Time-series analysis

## Dataset
Source: Our World in Data
https://ourworldindata.org/covid-deaths

## How to Run
1. Import schema.sql to create tables
2. Load CSV files into the tables
3. Run queries in the /queries folder in order

## Tools Used
- MySQL / PostgreSQL
